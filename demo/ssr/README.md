This folder demonstrates how to use React PDF viewer with a most basic SSR environment.

1. Install the dependencies

~~~
npm install
~~~

2. Build

~~~
npm run build
~~~

3. Running locally

~~~
npm run dev
~~~

Visit http://localhost:8001/. You can change the port `8001` in the `src/server.js` file: 

~~~ javascript
// src/server.js

const PORT = 8001;
~~~
